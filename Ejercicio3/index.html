Amazon Web Services (AWS)

Fecha de lanzamiento

25 de marzo de 2006

Empresa encargada de su desarrollo

Amazon.com, Inc.

Caracteristica mas atractiva de la aplicacion

AWS es una plataforma de servicios en la nube que ofrece una amplia gama de servicios como almacenamiento, cómputo, bases de datos y más. Una de sus características más innovadoras es la escalabilidad automática, que permite a las aplicaciones ajustar automáticamente los recursos según la demanda.

Tecnologias y herramientas implementadas en su desarrollo

  • Frontend: JavaScript, AngularJS, HTML, CSS
  • Backend: Java, Python, Go, Node.js
  • Base de datos: Amazon DynamoDB, Amazon RDS
  • Infraestructura: Infraestructura global de Amazon con centros de datos en todo el mundo
  • Otros: Amazon Lambda para computación sin servidor, Amazon S3 para almacenamiento de objetos

Facebook

Fecha de lanzamiento

4 de febrero de 2004

Empresa encargada de su desarrollo

Meta Platforms, Inc.

Caracteristica mas atractiva de la aplicacion

Facebook es una red social que conecta a personas alrededor del mundo, permitiéndoles compartir fotos, videos, mensajes y actualizaciones. Una de sus características más innovadoras es el algoritmo de personalización de feed, que utiliza machine learning para mostrar contenido relevante basado en los intereses del usuario.

Tecnologias y herramientas implementadas en su desarrollo

  • Frontend: React, JavaScript, HTML, CSS
  • Backend: PHP (HHVM), Hack
  • Base de datos: MySQL, TAO
  • Infraestructura: Meta's custom server architecture
  • Otros: GraphQL para consultas API, Memcached para caching

Netflix

Fecha de lanzamiento

29 de agosto de 1997 (como servicio de streaming desde 2007)

Empresa encargada de su desarrollo

Netflix, Inc.

Caracteristica mas atractiva de la aplicacion

Netflix es una plataforma de streaming de video que ofrece una amplia variedad de películas, series y documentales. Su característica más destacada es el algoritmo de recomendación de contenido, que analiza los hábitos de visualización de los usuarios para sugerirles contenido personalizado.

Tecnologias y herramientas implementadas en su desarrollo

  • Frontend: JavaScript, React, Node.js
  • Backend: Java, Python, Scala
  • Base de datos: Cassandra, MySQL, Elasticsearch
  • Infraestructura: AWS (Amazon Web Services)
  • Otros: Chaos Monkey para pruebas de resiliencia, CDN propio para distribución de contenido

Zoom

Fecha de lanzamiento

10 de septiembre de 2012

Empresa encargada de su desarrollo

Zoom Video Communications, Inc.

Caracteristica mas atractiva de la aplicacion

Zoom es una aplicación de videoconferencias que permite a los usuarios conectarse virtualmente desde cualquier lugar del mundo. Su característica más atractiva es la facilidad de uso y calidad de video, que permite reuniones sin interrupciones, incluso con conexiones de internet lentas.

Tecnologias y herramientas implementadas en su desarrollo

  • Frontend: JavaScript, React, HTML, CSS
  • Backend: Java, C++, Node.js
  • Base de datos: MySQL, Redis
  • Infraestructura: AWS, Azure
  • Otros: WebRTC para comunicaciones en tiempo real, Zoom SDK para integraciones de terceros

Google Drive

Fecha de lanzamiento

24 de abril de 2012

Empresa encargada de su desarrollo

Google LLC

Caracteristica mas atractiva de la aplicacion

Google Drive es una plataforma de almacenamiento en la nube que permite a los usuarios almacenar, compartir y colaborar en archivos desde cualquier dispositivo. Una de sus características más atractivas es la colaboración en tiempo real, que permite a múltiples usuarios editar un documento simultáneamente.

Tecnologias y herramientas implementadas en su desarrollo

  • Frontend: JavaScript, HTML, CSS
  • Backend: Python, Java
  • Base de datos: Bigtable, Spanner
  • Infraestructura: Google Cloud Platform (GCP)
  • Otros: OAuth 2.0 para autenticación, API de Google Drive
